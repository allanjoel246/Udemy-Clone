# Udemy Clone
https://allanjoel246.github.io/Udemy-Clone/

This is a clone of the Udemy-style course listing site built purely with HTML and CSS.

## Features  
- Home page showing featured courses with images, titles, instructors, ratings, and original vs discounted prices.  
- Sections for “Topics Recommended for You” (e.g., Website, JavaScript, CSS, Mongo DB, Azure, AWS, Docker, GitHub, React JS) and “Most Popular” courses.  
- Responsive layout (using CSS) so it works nicely on different screen sizes.  
- Clean and modern design inspired by Udemy’s look and feel.

## Technologies  
- HTML5  
- CSS3 (flexbox/grid for layout, hover effects, etc.)  
- No JavaScript (pure static site)
